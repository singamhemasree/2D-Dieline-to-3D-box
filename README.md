# 2D Dieline → 3D Box Folding

An interactive 3D box that folds from a flat 2D dieline using React + Three.js

## Features
- Drag slider to fold from 2D to 3D
- Sky texture with Sun, Moon, Clouds, Stars
- "HEMASREE" text on box
- Orbit controls to rotate

## Tech Stack
React + @react-three/fiber + @react-three/drei + Three.js

## How to Run
1. npm install
2. npm run dev
3. Open localhost:5173
